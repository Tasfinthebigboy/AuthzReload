Authz Bot Commands

Below is a list of all available commands for the Authz Bot.

Commands

Admin Commands

  • /verify [role] - Setup Verification system
  • /kick [user] [reason] - Kicks a user
  • /ban [user] [reason] - Bans a user
  • /unban [user] - Unbans a user
  • /mute [user] [reason] - Mutes a user
  • /unmute [user] - Unmutes a user
  • /warn [user] [reason] - Warns a user
  • /warnings [user] - Shows warnings of a user

Mod Commands

  • /warn [user] [reason] - Warns a user
  • /mute [user] [reason] - Mutes a user
  • /unmute [user] - Unmutes a user
  • /warnings [user] - Shows warnings of a user

Member Commands

  • /help - List of commands
  • /customembed [title] [description] - Create a custom embed
