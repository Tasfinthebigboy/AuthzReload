My Discord Bot Commands

Commands

Admin Commands

  • !verify @Role - Sets up a verification system.
  • !ban @User [reason] - Bans a user from the server.
  • !unban User#1234 - Unbans a user from the server.

Mod Commands

  • !kick @User [reason] - Kicks a user from the server.
  • !mute @User [reason] - Mutes a user in the server.
  • !unmute @User - Unmutes a user in the server.
  • !warn @User [reason] - Warns a user in the server.
  • !warnings @User - Shows warnings of a user.

Member Commands

  • !help - Shows the help menu.
